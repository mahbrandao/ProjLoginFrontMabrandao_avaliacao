# Primeiro Projeto Full Stack 

### Sobre o projeto 
Esta API foi desenvolvida como parte do meu primeiro projeto **Full Stack**, sendo responsável por toda a lógica de negócio, manipulação de dados e comunicação com o front-end.

A aplicação permite o cadastro e gerenciamento de usuários e produtos.
