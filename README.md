# Primeiro Projeto Full Stack

## Sobre o Projeto

Este projeto foi desenvolvido como meu primeiro projeto **Full Stack**.  
O front-end foi criado utilizando **Next.js**, sendo responsável pela interface visual da aplicação e pela conexão com minha API back-end.

A aplicação permite o cadastro e gerenciamento de usuários.
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

   **API do projeto:**  
[Clique aqui para acessar](https://github.com/mahbrandao/ProjLoginMabrandao_avaliacao.git)
